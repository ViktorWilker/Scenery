# Scenery — Android

Android app that generates immersive soundscapes from natural language descriptions. The user describes a scene and the app composes a layered sound mix for it, with scene name and mood generated by AI.

## What it does

The user types a scene: "dense forest after the rain", "train cabin crossing Siberia at night" and the app returns a sound composition with multiple layers playing simultaneously, each with a volume calibrated by foreground, midground, and background role.

## Architecture

Composition logic lives entirely in the backend. The app is responsible for capturing the prompt, displaying the generated scene, and managing audio layer playback.

```
br.com.scenery/
├── audio/
│   └── SoundManager          # manages simultaneous layer playback via MediaPlayer
├── data/
│   ├── local/                # local persistence
│   ├── model/
│   │   ├── AudioLayer        # audio layer with volume
│   │   ├── SceneResponse     # API response (scene_name, mood, sounds)
│   │   └── SoundLayer        # individual sound with id and preview_url
│   └── remote/
│       ├── GeminiApi.kt      # Retrofit interface for the backend
│       └── SceneRepository   # repository abstracting remote calls
├── ui/
│   ├── screen/               # Jetpack Compose screens
│   └── theme/                # app design system
├── viewmodel/                # UI state and layer orchestration
├── MainActivity
└── NavGraph.kt               # screen navigation
```

## Data flow

```
UI (screen) → ViewModel → SceneRepository → GeminiApi (Retrofit)
                                                    |
                                              POST /scene
                                                    |
                                             SceneResponse
                                                    |
                              ViewModel → SoundManager → MediaPlayer (per layer)
```

`SoundManager` starts one `MediaPlayer` per audio layer, each at the volume defined by the backend. Layers play simultaneously in loop.

## Stack

Kotlin · Jetpack Compose · Retrofit · Moshi · MediaPlayer · MVVM

Backend -> https://github.com/ViktorWilker/Scenery-backend
